# Automatically configure Linux security policies and take screenshots

这是一个用于配置Linux服务器安全策略并且截图的脚本，运行平台为Windows，使用Python3.9编写。
适用于东部某省的电力二次设备加固报告的编写。

需求环境python3.9+，Windwos10用户需要额外下载一个terminal，在MS store下载。

功能还需要进行优化和测试。
